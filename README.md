# 🚗 רובוט עסקאות רכב - יד2

רובוט שסורק יד2 כל 30 דקות ומשלח התראת וואטסאפ על רכבים מתחת למחיר שוק.

---

## הגדרה - 5 דקות בלבד

### שלב 1 - העלה ל-GitHub
1. צור ריפוזיטורי חדש ב-GitHub (פרטי)
2. העלה את שני הקבצים:
   - `car_bot.py`
   - `.github/workflows/car_bot.yml`

### שלב 2 - הגדר וואטסאפ (CallMeBot)
1. שלח `I allow callmebot to send me messages` למספר: **+34 644 59 47 64** בוואטסאפ
2. תקבל API Key בחזרה
3. ב-GitHub Repo → **Settings → Secrets → Actions** הוסף:
   - `CALLMEBOT_PHONE` = המספר שלך (+972...)
   - `CALLMEBOT_APIKEY` = הקוד שקיבלת

### שלב 3 - הגדר חיפוש
פתח `car_bot.py` ועדכן:

```python
# שנה את אחוז ההנחה המינימלי
DISCOUNT_THRESHOLD = 15   # 15% מתחת למחירון

# הכנס URLs מיד2 - כנס, סנן מה שאתה רוצה, העתק URL
YAD2_SEARCH_URLS = [
    "https://www.yad2.co.il/vehicles/cars?manufacturer=14&year=2019-2023",
    "https://www.yad2.co.il/vehicles/cars?manufacturer=56&price=40000-100000",
]
```

**איך לקחת URL מיד2:**
- כנס ל-yad2.co.il/vehicles/cars
- סנן: יצרן, שנה, מחיר, אזור - מה שאתה רוצה
- העתק את ה-URL מהדפדפן
- הדבק ב-YAD2_SEARCH_URLS

### שלב 4 - הפעל
- GitHub Actions יפעל אוטומטית כל 30 דקות
- לבדיקה מיידית: **Actions → Car Deal Bot → Run workflow**

## שינוי הגדרות

| פרמטר | ברירת מחדל | תיאור |
|--------|-----------|-------|
| `DISCOUNT_THRESHOLD` | 15 | % הנחה מינימלי ממחירון |
| `cron: "*/30 * * * *"` | 30 דקות | תדירות סריקה |

לסריקה כל 15 דקות שנה ל: `*/15 * * * *`

## פייסבוק מרקטפלייס

פייסבוק חוסמת סריקה אוטומטית. הפתרון הטוב ביותר:
- הגדר **התראת חיפוש** ישירות באפליקציית פייסבוק
- Marketplace → Search → Set Alert
- פייסבוק תשלח לך התראה מיידית בנייטיב
